HOW TO UPLOAD FILESHOME

Here are some instructions on how to upload the files

  • The first Excel file to be uploaded is a partially filled excel file.Each row represents a class, with lab hours and courses offered by other departments already fixed.
  • The second excel file to be uploaded is a Course-teacher mapped file.It has 2 columns and the first row should contain course and faculty as its values.The other rows can have course name and faculty name as its values.
  • The third excel file to be uploaded is a Course-hour mapped file.It has 2 columns and the first row should contain course and hour as its values.The other rows can have course name and number of hours required for that course per week as its values.
  • If you have any query related to uploading the input excel files please do refer the following sample excel files.
